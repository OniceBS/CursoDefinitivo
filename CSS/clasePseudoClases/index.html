Notas clase Pseudo clases y Pseudo elementos:

Pseudo clases
:class
Pseudo elementos
::element

Las Pseudo clases Definen un estado especial de un elemento. ejemplo :

.sect-nav a:hover {
color: red;
}

Los Pseudo elementos definen el estilo de una parte especifica de un elemento. ejemplo:

.sect-nav a::after {
content: " | ";
}
